# POST LOG — DASHBOARD (Table view on post)

## Columns

- posted_at
- type
- channel
- views
- comments
- clicks
- joins
- link
- (optional) saves, shares, dms_count

## Filters (save both)

- Last 7 days
- Last 30 days

## Sort

- posted_at desc

## Benchmarks (edit to taste)

- **CTR** (clicks / views) ≥ 1.5%
- **Join rate** (joins / clicks) ≥ 10%
- **Engagement** (comments / views) ≥ 0.5%
